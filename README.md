# Mon Journal — PWA

## Installation sur Android

1. Décompresse le fichier ZIP
2. Héberge les fichiers sur GitHub Pages (gratuit) :
   - Crée un compte sur github.com
   - Nouveau dépôt → Upload tous les fichiers
   - Settings → Pages → Branch: main → Save
   - Ton app sera disponible sur : https://TON_PSEUDO.github.io/NOM_REPO

3. Sur Android (Chrome) :
   - Ouvre l'URL de ton GitHub Pages
   - Menu ⋮ → "Ajouter à l'écran d'accueil"
   - L'app s'installe comme une vraie application !

## Fonctionnalités

- 🔒 Protection par code PIN à 4 chiffres
- 📝 Écriture avec titre optionnel
- 🔍 Recherche dans toutes les entrées
- 🌙 Thème sombre intégral
- 📅 Date et heure automatiques
- 💾 Stockage local (aucun serveur, données privées)
- 📱 Fonctionne hors connexion (Service Worker)

## Données

Toutes les entrées sont stockées localement dans le navigateur (localStorage).
Aucune donnée n'est envoyée sur internet.
